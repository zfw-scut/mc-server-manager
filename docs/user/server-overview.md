# 服务器概况

## 游戏服务器

- 名称：你好新蒸程
- 整合包版本：1.5.9
- Minecraft：1.21.1
- 连接地址：`wzk.rainplay.cn:56903`

客户端必须使用与服务端匹配的整合包。是否能够完成登录还会受到正版验证服务、客户端配置和模组注册表一致性的影响。

## 状态面板 Demo

- 地址：`http://wzk.rainplay.cn:56904/`
- 用途：只读查看服务状态、玩家数量和基础资源信息。
- 登录凭据：向服主获取，不记录在 GitHub。

当前面板仍是 Demo，使用 HTTP Basic Auth 且没有 HTTPS，不应在不可信网络中输入长期复用的密码。
